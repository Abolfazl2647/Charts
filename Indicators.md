### Volume indicator

Volume is added by default if the financial instrument supports it ([has_no_volume](https://github.com/Abolfazl2647/Charts/blob/main/Symbology#has_no_volume) is `true`). This behavior can be disabled using [create_volume_indicator_by_default](https://github.com/Abolfazl2647/Charts/blob/main/Featuresets).
